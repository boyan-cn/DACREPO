# 数组理论基础

1. 数组元素是不能删的，只能覆盖（删除操作也是从后往前移动覆盖，把最后一个元素覆盖成null/-1之类的约定无意义值）

2. 数组在内存中的存储方式（Java）：

   - 一维数组：连续内存地址空间，两个元素开始地址之间差一个元素占用的字节数(byte)；

   - 二维数组：行之间是随机地址，行内一维数组连续地址空间；
