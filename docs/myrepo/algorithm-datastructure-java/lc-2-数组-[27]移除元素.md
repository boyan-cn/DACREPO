## 题解

### 解法1：暴力解法

> 会超出时间限制，目的是练习基本功；

#### 解题思路

两个for循环嵌套，第一个循环：扫描，第二个循环：移动覆盖删除元素；

#### 复杂度

- 时间复杂度：O(n^2)
- 空间复杂度: O(1)，原地算法；

------

### 解法2：快慢双指针法

#### 题目中的关键点

- 原地**修改**输入数组
- 数组元素的顺序**可以改变**
- **不需要**考虑数组中超出新长度后面的元素

#### 解题思路

（快慢双指针法）

1. **快指针**：寻找新数组的元素，新数组就是不含有目标元素的数组。找到属于新数组的元素后，赋值给慢指针指向的新数组尾端，从而添加到新数组中，然后继续扫描下一个，快指针扫描到数组尾端停止循环；
2. **慢指针**：指向更新 新数组 后一个的位置 = 新数组.length；

## Code

### 解法1：暴力解法

```java
// 暴力解法，锻炼代码实现能力，会超时
class Solution {
    public int removeElement(int[] nums, int val) {
        // 准备辅助变量, “指向”数组下标
        int size = nums.length; 
        // 暴力遍历搜索 = val 的数组元素
        for (int i = 0; i < nums.length; i++) {   
            //  如果找到
            if (nums[i] == val) {
                // 覆盖删除
                for (int j = i + 1; j < nums.length; j++) {
                    nums[j - 1] = nums[j];
                }
                i--;    // 扫描指针回退一位：因为本来i指向的元素被删除，代替它的是一个新元素，需要重新比对；
                size--; // 更新数组长度
            }
        }
        // 返回结果
        return size;
    }
}     
// 时间复杂度：O(n^2)
// 空间复杂度：O(1)，原地算法
```

### 解法2：快慢双指针法

```java
// 快慢双指针法
class Solution {
    public int removeElement(int[] nums, int val) {
        // 初始化辅助“指针”变量
        int fast = 0, slow = 0;
        // 快慢指针：扫描，更新(不含val的新数组)
        for (; fast < nums.length; fast++) {
            // 扫描到新数组元素
            if (nums[fast] != val) {
                nums[slow] = nums[fast];
                slow++; // 插入后slow指针也后移一位
            }
        }
        // 返回 slow = 新数组.length
        return slow;
    }
}     
// 时间复杂度：O(n)
// 空间复杂度：O(1)，原地算法
```
